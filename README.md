# Sales Performance Dashboard (Excel)

## Project Overview
This project is an interactive Sales Performance Dashboard built using Microsoft Excel. It provides key business insights by analyzing sales, profit, quantity, return rate, and revenue after shipment.

## Features
- KPI Cards
  - Total Revenue
  - Total Quantity
  - Total Return Rate
  - Revenue After Shipment

- Interactive Slicers
  - Region
  - Product
  - Year

- Charts
  - Revenue by Region
  - Revenue Contribution by Product
  - Monthly Revenue Trend

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Excel Formulas
- Dashboard Design

## Key Insights
- Compare revenue across regions.
- Analyze product-wise revenue contribution.
- Monitor monthly revenue trends.
- Filter results dynamically using slicers.
